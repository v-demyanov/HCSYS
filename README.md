# HCSYS – .NET test task

HCSYS – Health Care System.

## Getting Started
This is the short guide on how to deploy the application locally using docker.

Before deploying you have to install such dependencies as: [Docker][docker-url] and [Makefile][makefile-url]. Then follow steps below.
- Create `.env` file in [root directory](./) using [.env.template](./.env.template)
- Execute command `make up` from [root directory](./)

[docker-url]: https://www.docker.com/
[docker-compose-url]: https://docs.docker.com/compose/
[makefile-url]: https://www.gnu.org/software/make/
